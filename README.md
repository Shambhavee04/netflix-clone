# 🎬 Netflix Clone

## 📌 Overview

This is a **Netflix Clone project** built using HTML, CSS, JavaScript, and Node.js.
It displays movies dynamically using an API and mimics the Netflix UI.

---

## 🚀 Features

* 🎥 Trending Movies
* ⭐ Top Rated Movies
* 🎬 Action Movies Section
* 🖼️ Dynamic Banner
* ▶️ Movie Click Functionality
* 🌐 Backend API Integration

---

## 🛠️ Tech Stack

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend:**

* Node.js
* Express.js

**API:**

* TMDB (The Movie Database)

## ⚙️ Setup Instructions

### 1. Clone the repo

git clone https://github.com/your-username/netflix-clone.git

### 2. Install dependencies

npm install

### 3. Run server

node server.js

---

## ▶️ How to Use

* Open `index.html` in browser
* Make sure backend is running
* Browse movies and click to view details

---

## ⚠️ Known Issues
